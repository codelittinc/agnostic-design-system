## Getting Started

```javascript
import React, { Component, useState } from 'react';
import { Dropdown } from '@codelittinc/agnostic-design-system';

class SimpleDropdown extends Component {
  const [value, setValue] = useState('');
  const options = [
    {
      label: 'Option 1',
      value: '1'
    },
    {
      label: 'Option 2',
      value: '2'
    }
  ];

  render() {
    return (
      <Dropdown
        id='ads-dropdown'
        value={value}
        options={options}
        onChange={item => {
          setValue(item ? item.value : '');
        }}
        getItemLabel={item => item.label}
        getItemKey={item => item.value}
        getItemValue={item => item.value}
      />
    );
  }
}

export default SimpleDropdown;
```

# Customazing styles

For the [Dropdown](https://design-system.codelitt.dev/?path=/story/components-dropdown--default-dropdown) component we use the following [naming convention](https://design-system.codelitt.dev/?path=/story/welcome-customizing-styles--page) to configure the CSS variables:

**--dropdown-[property]?-[attribute]-[state]?**

- Property - icon, arrow-icon
- Attribute - background-color, border, font, etc (required)
- State - hover

**Important:** The Dropdown component is composed by 2 other components: [Input](https://design-system.codelitt.dev/?path=/docs/components-input--default) and [List](https://design-system.codelitt.dev/?path=/docs/components-list--simple-list), you will be able to use their variables to customize this component.

1. Create the css class using our variables

```css
.my-dropdown-css {
  /* Component + Attribute */
  --dropwdown-border-radius: 1px solid grey;

  /* Component + Attribute */
  --input-background-color: white;

  /* Component + Property + Attribute */
  --input-placeholder-color: grey;

  /* Component + Attribute */
  --list-border: 1px solid #a2cee1;

  /* Component + Attribute + State*/
  --list-item-border-bottom-hover: 1px solid red;
}
```

2. Pass the css class through its `variablesClassName` property

`<Dropdown variablesClassName={styles['my-dropdown-css']} />`
