# Tag Component

## Type: Simple

## Tag Props

|        Prop        |      Type       | Default | Mandatory |
| :----------------: | :-------------: | :-----: | :-------: |
|      children      | React.ReactNode |    -    |   true    |
| variablesClassName |     string      |    -    |   false   |

## Getting Started

```javascript
import React from 'react';
import { Tag } from '@codelittinc/agnostic-design-system';

const MyComponent = () => {
  return (
    <div>
      <Tag>Tag Name</Tag>
    </div>
  );
};

export default MyComponent;
```
