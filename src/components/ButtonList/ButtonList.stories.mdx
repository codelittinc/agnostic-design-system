# ButtonList Component
The ButtonList component is based on the [Button](https://design-system.codelitt.dev/?path=/story/components-button) component

## ButtonList Props

|        Prop        |                    Type                    | Default | Optional |
| :----------------: | :----------------------------------------: | :-----: | :------: |
|         id         |                   string                   |    -    |  false   |
|       getItem      |        (item: T) => React.ReactNode        |    -    |  false   |
|      getItemKey    |             (item: T) => string            |    -    |  false   |
|      onChange      |                (e: T) => void              |    -    |  false   |
|       options      |                    T[]                     |    -    |  false   |
|       selected     |                     T                      |    -    |   true   |
| variablesClassName |                   string                   |    -    |   true   |

## Getting Started

```javascript
import React, { Component, useState } from 'react';
import { ButtonList } from '@codelittinc/agnostic-design-system';

const MyButtonList = () =>{
  const [selectedItem, setSelectedItem] = useState();

  const options = [
    { name: 'Software', id: 'Software' },
    { name: 'Financing', id: 'Financing' },
    { name: 'Retail', id: 'Retail' },
    { name: 'Pet', id: 'Pet' },
    { name: 'Law', id: 'Law' }
  ];

  render() {
    return (
      <ButtonList
        onChange={setSelectedItem}
        selected={selectedItem}
        options={options}
      />
    );
  }
}

export default MyButtonList;
```
