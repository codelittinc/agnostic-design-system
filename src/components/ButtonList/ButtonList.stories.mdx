# ButtonList Component

## Type: Decorator

The Button List is based on the [Button](https://design-system.codelitt.dev/?path=/story/components-button) component

## Button List Props

|        Prop        |                                         Type                                          |                                              Description                                              |        Default         | Mandatory |
| :----------------: | :-----------------------------------------------------------------------------------: | :---------------------------------------------------------------------------------------------------: | :--------------------: | :-------: |
|      category      | [Category](https://design-system.codelitt.dev/?path=/docs/components-button--neutral) | A button can have one of any six categories (neutral, primary, secondary, positive, negative, ghost). |       'neutral'        |   false   |
|         id         |                                        string                                         |                                              Unique id.                                               |           -            |   true    |
|      getItem       |                             (item: T) => React.ReactNode                              |                                         Accepts custom items                                          |           -            |   false   |
|     getItemKey     |                                  (item: T) => string                                  |                                              Item's key                                               |           -            |   true    |
|    getItemLabel    |                                  (item: T) => string                                  |     Text to be displayed in the Button, by default it expects an item with the property **name**      | (item: T) => item.name |   false   |
|      onChange      |                                    (e: T) => void                                     |                           Calls a function whenever the button is clicked.                            |           -            |   true    |
|      options       |                                          T[]                                          |                                            List of options                                            |           -            |   true    |
|      selected      |                                           T                                           |                                        Initially selected item                                        |           -            |   false   |
| variablesClassName |                                        string                                         |                                    Accepts custom CSS class names.                                    |           -            |   false   |

## Getting Started

```javascript
import React, { Component, useState } from 'react';
import { ButtonList } from '@codelittinc/agnostic-design-system';

const MyButtonList = () =>{
  const [selectedItem, setSelectedItem] = useState();

  const options = [
    { name: 'Software', id: 'Software' },
    { name: 'Financing', id: 'Financing' },
    { name: 'Retail', id: 'Retail' },
    { name: 'Pet', id: 'Pet' },
    { name: 'Law', id: 'Law' }
  ];

  render() {
    return (
      <ButtonList
        id={'button-list'}
        onChange={setSelectedItem}
        selected={selectedItem}
        options={options}
      />
    );
  }
}

export default MyButtonList;
```

We use the following [naming convention](https://design-system.codelitt.dev/?path=/story/welcome-customizing-styles--page). Please check the docs to see the variables you can customize.
