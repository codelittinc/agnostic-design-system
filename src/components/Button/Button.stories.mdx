# Button Component

## Button Props

|        Prop        |      Type       | Default | Optional |
| :----------------: | :-------------: | :-----: | :------: |
|     appendIcon     | React.ReactNode |    -    |   true   |
|      content       | React.ReactNode |    -    |   true   |
|      disabled      |     boolean     |  false  |   true   |
|       label        |     string      |    -    |   true   |
|      category      |    Category     | neutral |   true   |
|       fixed        |     boolean     |  false  |   true   |
|      onClick       |   () => void    |    -    |  false   |
|        size        |      Size       |  large  |   true   |
| variablesClassName |     string      |    -    |   true   |
|   withAppendIcon   |     boolean     |  false  |   true   |

---

|     Props      | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| :------------: | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **appendIcon** | By default it receives an [Arrow Icon](https://design-system.codelitt.dev/?path=/story/components-button--with-append-icon), but it is possible to send your own [Icon](https://design-system.codelitt.dev/?path=/story/components-button--with-custom-append-icon). The Icon can also be the [only element inside the button](https://design-system.codelitt.dev/?path=/story/components-button--without-label-with-append-icon), if there isn't a value passed to the label. |

## Getting Started

```javascript
import React, { Component } from 'react';
import { Button } from '@codelittinc/agnostic-design-system';

class SimpleButton extends Component {
  const handleClick = () => {
    console.log('Button clicked!');
  }

  render() {
    return (
      <Button onClick={handleClick} />
    );
  }
}

export default SimpleButton;

```

# Customazing styles

For the Button component we use the following naming convention to configure the CSS variables:

**--button-[property]?-[attribute]-[state]?**

- Property - append-icon (optional)
- Attribute - background-color, border, font, etc (required)
- State - hover, focus, active, disabled (optional)

1. Create the css class using our variables

```css
.my-button-css {
  /* Component + Attribute */
  --button-background-color: red; /* Simplest variable */

  /* Component + Attribute + State */
  --button-background-color-hover: blue; /* Using state */

  /* Component + Property + Attribute */
  --button-append-icon-color: green;

  /* Component + Property + Attribute + State */
  --button-append-icon-color-active: red;
}
```

2. Pass the css class through its `variablesClassName` property

`<Button variablesClassName={styles['my-button-css']} />`
