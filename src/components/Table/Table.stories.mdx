# Table Component

## Type: Simple

## Table Props

|        Prop        |                   Type                   |                                   Description                                    | Default | Mandatory |
| :----------------: | :--------------------------------------: | :------------------------------------------------------------------------------: | :-----: | :-------: |
|     headerList     |     array of strings or react nodes      |            Specifies the list to be used to populate the table header            |    -    |   true    |
|     footerList     |     array of strings or react nodes      |            Specifies the list to be used to populate the table footer            |    -    |   false   |
|    renderFooter    |                 function                 |                      Returns the full footer HTML structure                      |    -    |   false   |
|    tableContent    | array of array of strings or react nodes | Each array inside the matrix represents a row in the table with all its content. |    -    |   true    |
| variablesClassName |                  string                  |             Specifies the custom class to apply styles to the table              |    -    |   false   |

## Getting Started

```javascript
import { Table } from '@codelittinc/agnostic-design-system';

const props = {
  headerList: ['First Name', 'Last Name', 'Email'],
  tableContent: [
    ['Ana', 'Robin', 'ana@mail.com'],
    ['Beth', 'Julius', 'beth@mail.com'],
    ['Ruth', 'Katherine', 'ruth@mail.com'],
    ['Julia', 'Mark', 'julia@mail.com'],
    ['Nath', 'Sloan', 'nath@mail.com']
  ]
};

const Table = props => {
  return (
    <Table
      headerList={props.headerList}
      tableContent={props.tableContent}
      variablesClassName='my-custom-table'
    />
  );
};

export default Table;
```
