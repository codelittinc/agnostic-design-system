## Getting Started

```javascript
import React, { useState } from 'react';
import { PhoneInput } from '@codelittinc/agnostic-design-system';

const SimplePhoneInput = () => {
  const [value, setValue] = useState('');

  return <PhoneInput onChange={phone => setValue(phone)} label='Phone Input' value={value} />;
};

export default SimplePhoneInput;
```

## CSS Variables Available

Given the nature of a decorator component, we use all the css variables of the **Input** component. 
If you need to specify or customize the input with your own class, the component will receive an inputVariablesClassName, which will allow you to overwrite the styles of our default input compontent.
