# RichTextEditor Component

## Type: Simple

## RichTextEditor Props

|         Prop         |   Type   |                              Description                              |   Default   | Mandatory |
| :------------------: | :------: | :-------------------------------------------------------------------: | :---------: | :-------: |
|   menuEditOptions    | string[] |            List of the edit options available on the menu.            | All options |   false   |
| menuPositionedBottom | boolean  | Specifies whether or not the menu bar should be under the text input. |      -      |   false   |
|    editorContent     |  string  |     Text to be displayed by default inside the editor container.      |      -      |   false   |
|   placeholderText    |  string  |        Text to be displayed as a placeholder on the container.        |      -      |   false   |
|  variablesClassName  |  string  |                    Accepts custom CSS class names.                    |      -      |   false   |

**Menu Edit Options**

1. blockquote
2. bullet list
3. code block
4. hard break
5. h1
6. h2
7. h3
8. h4
9. h5
10. h6
11. horizontal rule
12. ordered list
13. bold
14. code
15. italic
16. strike
17. underline
18. clear marks
19. clear nodes
20. undo
21. redo
22. table
23. image

## Getting Started

```javascript
import React from 'react';
import { RichTextEditor } from '@codelittinc/agnostic-design-system';

const SimpleRichTextEditor = () => {
  return (
    <RichTextEditor
      menuEditOptions={['blockquote', 'bullet list', 'code block', 'hard break', 'h1', 'redo']}
      editorContent='<h2>Edit here</h2><br><br>'
    />
  );
};

export default SimpleRichTextEditor;
```
