Input example:

Idle
```js
<Input
  label="Input Label"
  name="favorite_food"
  placeholder="Enter some data"
  value="This is a text value..."
/>
```

Success
```js
<Input
  label="Input Label"
  name="favorite_food"
  placeholder="Enter some data"
  status="success"
  statusText="This is a success message"
  type="text"
  value="This is a text value..."
/>
```

Error
```js
<Input
  label="Input Label"
  name="favorite_food"
  placeholder="Enter some data"
  status="error"
  statusText="This is an invalid value"
  type="text"
  value="This is a text value..."
/>
```
